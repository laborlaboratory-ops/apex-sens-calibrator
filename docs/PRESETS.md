## プリセット追加手順

### 1. データ収集
- 選手のスコープ別感度値（1x〜10x + アイアンサイト）を確認
- ソース（URL、日付）を記録
- ベース設定（数字感度、反応曲線、デッドゾーン）もメモ

### 2. PRESETS.mdに記録
このファイルに上記フォーマットで追加。

### 3. コードに追加

`src/App.jsx` の `PRESET_DATA` オブジェクトに追加:

```javascript
PRESET_DATA.newplayer = {
  raw: { "1x": 値, "2x": 値, "3x": 値, "4x": 値, "6x": 値, "8x": 値, "10x": 値 },
  iron: アイアンサイト値  // 不明な場合は1xと同じ値
};
```

`PRESETS` 配列に追加:

```javascript
{ id: "newplayer", name: "表示名", desc: "説明文", tag: "プロ" }
```

tagの種類: `計算` / `プロ` / `配信者` / `研究者` / `参考`

比率計算（makeRatios）とIRON_RATIOSは自動で処理されるため手動計算は不要。

### 4. 確認
- グラフで他のプリセットと比較表示できること
- cfg出力のscalar_7がiron値で正しく出力されること
